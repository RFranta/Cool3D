RepRap Parts

When tagging parts, tag exactly as "part for <exact name>".
The back end will add underscores.
For example, "part for Darwin" becomes "part_for_Darwin".
This is a simple tagging process, but should prevent accidental tagging.
Also, remember that you can tag other people's objects if you see they've done it incorrectly.
